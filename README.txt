Fantasy Escape is a Rougelikegame where the player has to reach the end of the level while eliminating enemies using various items to help player progress through the level.

----------------------------------------
Main features of the game:
----------------------------------------
A functional main menu that is easy to navigate.
Player character that can move around and kill/avoid enemies.
Currency acquired by picking up currency.
High Score determined by time taken to finish a level
Open up chest scattered around the levels that generate a random item from the item pool(heart, sword bow, etc.).
Player character uses items to help progress through the level.
Player character only has 3 stocks for the rest of the game and heart item increases it by one. If the player dies in level 2 onwards, the player has to restart from level 1. so for example if the player reaches level 7 but dies during that level they have to restart from level 1 with no items.
The player can see their in-game score and the current amount of currency they have acquired.
There will be different types of enemies the player has to defeat, some of them will have ranged attacks and debuffs that affect the player.


----------------------------------------
Controls:
----------------------------------------

Move - Arrow keys or WASD
Jump - Space
Attack - J
Dash - K
